#Test-driven Development in JS

Test-driven development cycle
``````````````````````````````
1. Write test
2. Run test (expect FAILURE)
3. Write code
4. Run test again (expect SUCCESS)
5. Refactor code
6. Repeat steps (until test is SUCCESSFULL)

* Sometimes called RED, GREEN REFACTOR
